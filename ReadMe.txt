WorkSphere – Visual Staff Management

Mohamed Oughlane



WorkSphere est une application web qui permet de gerer le personnel directement sur un plan d’etage interactif. L’objectif est d’organiser, deplacer et controler les employes selon leurs roles, avec une interface simple, moderne et responsive.

Fonctionnalites cles:

Ajout, déplacement et suppression d’employes

Modale d’ajout avec previsualisation photo + experiences dynamiques

Zones du plan : Reception, Salle des serveurs, Securite, Conference, Personnel, Archives

Restrictions d’acces selon les roles

Profil complet au clic (photo, infos, experiences, position)

Indicateurs visuels (zones vides, limites par zone)

Interface Desktop & Mobile (Flexbox, Grid)

Regles principales:

Reception → Receptionnistes

Serveurs → IT

Securite → Agents de securite

Managers → Acces total

Nettoyage → Partout sauf Archives

Autres → Acces hors zones restreintes



Gestion du projet:

Planification : Trello / Jira / GitHub Projects

Deploiement : GitHub Pages ou Vercel

Validation : W3C + responsive complet
